# Warframe EE.log 日志

## 目录

`C:\Users\{你的用户名}\AppData(隐藏)\Local\Warframe\EE.log`

## 玩家信息

| 关键词 | 内容 |
| --- | --- |
| public address | 本机IP地址 |
| host address | 加入的主机IP地址 |
| AddSquadMember | 玩家的名字和IP |
| Discarding | 丢包的IP |
| MissionInfo | 任务信息 |
| PRIVMSG | 发送的消息 |
| was downed | 倒地日志 |
| was killed | 死亡日志 |

## 夜灵 BOSS

| 关键词 | 内容 |
| --- | --- |
| Weakpoint Destroyed | 关节破坏 (17.186) |
| A shard has been put in the Eidolon Shrine | 碎片放入 |
| A shard has been removed from the Eidolon Shrine | 碎片取出 |
| Teralyst spawned | 夜灵 BOSS 生成 (1) |
| Eidolon spawning SUCCESS | 碎片后的 夜灵 BOSS 生成 (2,3) |
| Teralyst Captured | 捕获 |
| Teralyst Killed | 击杀 (不含动画击杀/档案击杀) |
