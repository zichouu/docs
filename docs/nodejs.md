# Node.js
[官网](https://nodejs.org/en/download)

## npm 淘宝源
```
npm config set registry https://registry.npmmirror.com
```

## yarn 淘宝源
```
yarn config set registry https://registry.npmmirror.com
```

## 全局
::: code-group
``` [添加]
npm i -g <name>
```
``` [移除]
npm uninstall -g <name>
```
:::

## 添加依赖
::: code-group
``` [npm]
npm i <name>
```
``` [yarn]
yarn add <name>
```
:::

## 移除依赖
::: code-group
``` [npm]
npm uninstall <name>
```
``` [yarn]
yarn remove <name>
```
:::
